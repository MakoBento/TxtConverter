# TxtConverter

各種ファイルを `.txt` 形式に変換するツールです。

## 機能

- **Excel ファイル（.xlsx / .xlsm / .xls）** の各シートのセル内容をタブ区切りテキストに変換
  - 取り消し線が付いた文字は `<deleted>...</deleted>` タグで囲んで出力
  - `.xls` は xlrd を使用（取り消し線判定なし）
- **テキスト系ファイル** の内容をそのまま `.txt` として出力
  - 文字コードは UTF-8 → CP932（Shift-JIS）の順にフォールバック

## 対応ファイル形式

| 種別 | 拡張子 |
|------|--------|
| Excel | `.xlsx` `.xlsm` `.xls` |
| テキスト系 | `.cs` `.md` `.txt` `.sql` `.py` `.js` `.html` `.htm` `.ts` `.tsx` `.css` `.vue` `.json` `.xml` |

## 使用方法

### ビルド

`build.bat` を実行すると `dist\TxtConverter.exe` が作成されます。

```
build.bat
```

### 操作手順

1. **対象ファイル / フォルダ** を指定する
   - 「ファイル参照」ボタンでファイルを選択
   - 「フォルダ参照」ボタンでフォルダを選択（サブフォルダ内のファイルも対象）
   - テキストボックスへのドラッグ&ドロップも可能
2. **出力先フォルダ** を指定する（「参照...」ボタンまたはドラッグ&ドロップ）
3. **「変換実行」** ボタンをクリック

### 出力ファイル名

```
元ファイル名_yyyyMMddHHmmss.txt
例: main.py_20260322120000.txt
```

## 動作環境

- Python 3.x
- 依存ライブラリ: `requirements.txt` 参照（openpyxl, xlrd, tkinterdnd2 など）
